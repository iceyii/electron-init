# electron-init

## 简介

Electron + Vue3 + Vite + Element-Plus + IndexDb快速开发模板

简洁高效, 无需调试, 一键启动, 马上上手开发

- 最新版Electron, 桌面APP利器
- Vue3 + Vite2开发速度拉满
- Element-Plus优雅秀气
- IndexDb本地数据库, 封装API, 即刻上手
- 封装常用组件等

## 1. 安装

```shell
npm install
或
yarn
```

## 2. 开发

```shell
npm run dev
```

## 3. 打包

```shell
npm run build
```
